## Project: Sales Performance Dashboard (Power BI)

## Objective

To build a clear and interactive dashboard for sales representatives to track performance, analyze trends, and estimate commissions.

## Problem

Sales data was stored in Google Sheets with manual tracking, making it difficult to analyze performance, compare periods, and forecast commissions.

## Solution

Developed a Power BI dashboard that provides individual views for each salesperson with dynamic filters and detailed analytics.

## Key Features

* Individual dashboard view for each salesperson
* Filters: Month, Year, Fiscal Year (July–June)
* Year-over-Year comparison
* Top clients and client type analysis
* Top-selling products
* Department-wise performance tracking
* Drill-down: View all orders for selected client
* Commission forecast based on predefined logic
* Clean and interactive visual design

## Tech Stack

* Power BI (.pbix)
* Google Sheets (data source)

## Business Impact

* Improved visibility of sales performance
* Faster decision-making
* Easy tracking of commissions and trends
